# Exploratory Data Analysis (EDA) Notebooks for "Exploración y Análisis de Datos para la Clasificación de Imágenes utilizando el Espacio de Características".

This repository contains a series of Jupyter notebooks performing **Exploratory Data Analysis (EDA)** for the project proposal titled **Exploración y Análisis de Datos para la Clasificación de Imágenes utilizando el Espacio de Características**. The goal of this project is to compare the effectiveness of traditional and modern feature extraction methods for image classification tasks. The EDA process aims to understand the dataset, explore its characteristics, and uncover any patterns or insights that can guide the subsequent phases of the project.
