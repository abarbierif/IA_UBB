# Exploratory Data Analysis (EDA) Notebooks for "Análisis Comparativo de Métodos de Extracción de Características y su Impacto en el Espacio de Características para la Clasificación de Imágenes".

This repository contains a series of Jupyter notebooks performing **Exploratory Data Analysis (EDA)** for the project proposal titled **Análisis Comparativo de Métodos de Extracción de Características y su Impacto en el Espacio de Características para la Clasificación de Imágenes**. The goal of this project is to compare the effectiveness of traditional and modern feature extraction methods for image classification tasks. The EDA process aims to understand the dataset, explore its characteristics, and uncover any patterns or insights that can guide the subsequent phases of the project.
