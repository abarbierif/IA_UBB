# TicTacToe

Here are all the files related to the TicTacToe project.

### The recommended setup is:

go to the repository and switch branch:

`cd <IA_UBB_path>`

`git checkout TicTacToe`

create a venv:

`python3 -m venv <venv_name>`

activate the venv:

`source <venv_name>/bin/activate` (Linux)

`.\<venv_name>\Scripts\activate.bat` (Windows CMD)

`.\<venv_name>\Scripts\activate.ps1` (Windows Power shell)

install the requirements:

`pip install -r requirements.txt`

play TicTacToe!!:

`python3 main.py` or `python3 TicTacToeAllInOne.py`
